# AstraOS Dashboard Refactored

A production-ready, clean widget dashboard setup.

## Running Locally

### Option 1: VS Code Live Server

1. Open the `astraos-dashboard` folder in VS Code.
2. Install the **Live Server** extension (by Ritwick Dey).
3. Right-click on `index.html` and select **Open with Live Server**.

### Option 2: Node static server

1. Ensure you have Node.js installed.
2. Install the `serve` package globally:
   ```bash
   npm install -g serve
   ```
3. Navigate to the `astraos-dashboard` folder:
   ```bash
   cd path/to/astraos-dashboard
   ```
4. Run the server:
   ```bash
   serve .
   ```
5. Open the locally hosted URL (usually `http://localhost:3000`) in your browser.
